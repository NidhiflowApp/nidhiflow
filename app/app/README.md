# Personal Budget Dashboard

A personal finance management dashboard built with React to track income, expenses, budgets, and investments with visual insights.

## ✨ Features (v1.0)

### 📊 Dashboard
- Monthly & Year-based selection
- Income, Expense, Investment summary
- Category-wise expense split
- Budget vs Actual comparison
- Paid-by analysis
- Top expense categories

### 📈 Charts Dashboard
- Monthly expense trend (last 6 months)
- Expense by category (selected month)
- Budget vs Actual (monthly)
- Paid By (monthly)
- Top Categories (monthly)
- Investment Overview (All-time)

## 🧱 Tech Stack
- React (Create React App)
- Chart.js + react-chartjs-2
- CSS (Glassmorphism UI)
- Git & GitHub
